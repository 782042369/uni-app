### 核心数据看板

**组件**: `CoreDataDashboard.vue`

**数据卡片配置**:

| 卡片类型 | 数据项 | 单位 | 颜色 |
|---------|--------|------|------|
| 生产核心数据 | 今日采煤量 | 万吨 | 黑色加粗 |
| 生产核心数据 | 今日剥离量 | 万立方米 | 黑色加粗 |
| 安全核心数据 | 未处理风险 | 项 | 橙色加粗（>0时） |
| 安全核心数据 | 未处理隐患 | 项 | 橙色加粗（>0时） |
| 设备核心数据 | 在线设备 | 台 | 绿色加粗 |
| 设备核心数据 | 故障设备 | 台 | 红色加粗（>0时） |
| 环保核心数据 | 水质/土壤/固废/大气 | 达标/异常 | 绿色/红色 |
| 产存销核心数据 | 今日产量 | 万吨 | 黑色加粗 |
| 产存销核心数据 | 当前库存 | 万吨 | 黑色加粗 |

**交互规则**:

1. 点击数据卡片跳转至对应详情页
2. 异常数据卡片边框显示警示色并轻微闪烁
3. 数据每 5 分钟自动刷新
4. 点击刷新图标手动刷新

### 待办提醒

**组件**: `TodoReminder.vue`

**数据结构**:

```typescript
interface TodoItem {
  id: string
  type: '隐患整改' | '预警处置' | '其他'
  content: string        // 不超过15字
  deadline: string       // 如"今日18:00前"
  status: 'pending' | 'processing' | 'completed'
  link: string           // 跳转链接
}
```

**交互规则**:

1. 最多显示 3 条待办
2. 左侧显示橙色警示点
3. 点击待办条目直接跳转至处理页面
4. 超出部分显示"更多待办"链接

### 常用功能快捷菜单

**组件**: `QuickActionGrid.vue`

**宫格配置**:

| 序号 | 功能名称 | 图标 | 跳转路径 |
|------|---------|------|---------|
| 1 | 隐患上报 | 警示符号 | `/safety/hazard?action=create` |
| 2 | 视频监控 | 摄像头 | `/production/video` |
| 3 | 风险查看 | 风险符号 | `/safety/risk` |
| 4 | 生产报表 | 报表符号 | `/production/report` |
| 5 | 应急通讯录 | 电话符号 | `/safety/contacts` |
| 6 | 设备状态 | 设备符号 | `/production/monitor` |
| 7 | 环保监测 | 绿叶符号 | `/safety/environmental` |
| 8 | 技术资料 | 书本符号 | `/knowledge` |
| 9 | 消息通知 | 铃铛符号 | `/profile/messages` |

**布局**: 3×3 宫格，每个宫格包含图标（24×24dp）+ 文字（不超过6字）

## 安全环保中心

**路由路径**: `/safety`
**二级菜单**: 6个（安全监测、预警处置、风险管控、应急通讯录、隐患排查、环保监测）

### 页面布局模板

```
┌─────────────────────────────┐
│      安全环保中心            │  标题栏
├─────────────────────────────┤
│[安全监测][预警处置][风险...]│  二级菜单（横向滑动）
├─────────────────────────────┤
│                             │
│        页面内容              │
│                             │
└─────────────────────────────┘
```

### 安全监测（默认显示）

**路由路径**: `/safety/monitor`

**筛选栏**: 横向滑动，包含 7 类监测类型

| 监测类型 | 监测内容 | 异常提醒 |
|---------|---------|---------|
| 人员监测 | 位置、在岗状态、防护装备 | 人员离岗、未佩戴防护装备 |
| 车辆监测 | 位置、速度、运行状态 | 车辆超速、偏离作业区域 |
| 边坡监测 | 位移、沉降、裂缝 | 边坡位移超标 |
| 煤层防自燃 | 温度、瓦斯浓度 | 温度超标、瓦斯浓度异常 |
| 防排水 | 水位、排水量 | 水位超标、排水故障 |
| 生产系统安全监测 | 运行状态 | 系统故障、运行异常 |
| AI视频监测 | 实时视频、智能识别 | 违规动火、人员聚集 |

**内容区域**: 默认显示地图模式，监测点按状态区分（正常-绿色、异常-红色、预警-橙色）

### 预警处置

**路由路径**: `/safety/warning`

**预警筛选**: 按预警级别、类型、处置状态筛选

**预警列表**: 每条预警包含预警级别（颜色区分）、预警内容、预警时间、发生位置

**预警级别颜色**:

| 级别 | 颜色 | 说明 |
|------|------|------|
| 一般 | 蓝色 #3B82F6 | 常规预警 |
| 较重 | 黄色 #F59E0B | 需要关注 |
| 严重 | 橙色 #F97316 | 需要立即处理 |
| 特别严重 | 红色 #EF4444 | 紧急情况 |

**处置流程**: 查看预警 → 分配处置 → 现场处置 → 反馈结果

### 风险管控

**路由路径**: `/safety/risk`

**风险分级筛选**: 按风险等级筛选

**风险等级颜色**:

| 等级 | 颜色 | 说明 |
|------|------|------|
| 重大风险 | 红色 #EF4444 | 最高风险级别 |
| 较大风险 | 橙色 #F97316 | 高风险级别 |
| 一般风险 | 黄色 #F59E0B | 中等风险级别 |
| 低风险 | 蓝色 #3B82F6 | 较低风险级别 |

**风险列表**: 每条风险包含风险名称、风险等级、管控责任人、管控措施、风险位置

**功能**: 风险上报（右上角"+"）、风险更新（管控责任人可更新）

### 应急通讯录

**路由路径**: `/safety/contacts`

**通讯录分类**: 横向滑动，包含应急指挥组、安全管理部、医疗急救、消防救援、设备维修、外部救援

**联系人列表**: 每条联系人包含姓名、部门、职务、联系方式

**功能**:
- 点击联系方式直接拨号
- 长按保存至手机通讯录
- 搜索联系人
- 紧急呼叫（一键拨打应急指挥组电话）

### 隐患排查

**路由路径**: `/safety/hazard`

**隐患筛选**: 按隐患等级、整改状态、排查区域筛选

**整改流程**: 上报隐患 → 分配整改 → 整改反馈 → 验收归档

**隐患状态颜色**:

| 状态 | 颜色 | 说明 |
|------|------|------|
| 未整改 | 红色 #EF4444 | 待处理 |
| 整改中 | 黄色 #F59E0B | 处理中 |
| 已整改 | 绿色 #10B981 | 已完成 |
| 验收不合格 | 橙色 #F97316 | 需重新整改 |

**功能**: 隐患上报、隐患跟踪、隐患统计

### 环保监测

**路由路径**: `/safety/environmental`

**监测类型**: 水质、土壤、固废、大气、噪声

**内容区域**: 默认显示数据图表（折线图/柱状图），下方显示实时监测数据

**监测指标**:

| 监测类型 | 监测指标 | 标准值 |
|---------|---------|-------|
| 水质监测 | pH值、COD、氨氮 | 国标限值 |
| 土壤监测 | 重金属含量、pH值 | 国标限值 |
| 固废监测 | 堆放量、堆放位置 | 堆放限值 |
| 大气监测 | PM2.5、PM10、二氧化硫 | 国标限值 |
| 噪声监测 | 噪声值 | 噪声标准 |

**功能**: 数据导出、异常提醒设置、离线缓存

## 生产运营

**路由路径**: `/production`
**二级菜单**: 5个（生产监控、综合自动化、生产报表、生产视频、产运销）

### 生产监控（默认显示）

**路由路径**: `/production/monitor`

**监控类型**: 作业车辆、无人驾驶

**内容区域**: 默认显示地图模式，车辆/设备按状态区分（正常运行-绿色、故障-红色、闲置-蓝色）

**车辆/设备详情**: 设备编号、运行状态、速度、作业区域、驾驶员信息

**功能**: 视频联动、设备调度、故障报警

### 综合自动化

**路由路径**: `/production/automation`

**自动化系统**: 破碎站无人值守、电力系统、运输系统自动化、排土系统自动化

**系统状态**: 系统名称、运行状态（正常、故障、待机）、运行参数、责任人

**功能**: 远程启停（仅管理员权限）、异常提醒

### 生产报表

**路由路径**: `/production/report`

**报表类型**: 采剥报表、运输报表、排土报表、产量汇总报表

**报表筛选**: 按时间筛选（今日、昨日、近7天、近30天、自定义时间）、按作业区域筛选

**报表格式**: 简洁表格（适配移动端，横向可滑动），下方显示汇总数据

**功能**: 报表导出、报表分享、数据对比

### 生产视频

**路由路径**: `/production/video`

**视频区域**: 作业面、破碎站、运输线路、排土场、办公区

**视频列表**: 区域名称、摄像头编号、实时画面缩略图、在线状态

**视频播放**: 全屏播放、倍速播放、暂停、抓拍、回放（近24小时）

**功能**: 视频联动、摄像头切换、弱网适配

### 产运销

**路由路径**: `/production/sales`

**数据类型**: 产量、销售、库存

**数据展示**: 采用图表（柱状图）+ 表格结合方式

**功能**: 数据导出、库存预警设置、销售明细查询

## 知识库

**路由路径**: `/knowledge`
**二级菜单**: 无

```
┌─────────────────────────────┐
```
